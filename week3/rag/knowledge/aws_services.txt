Amazon Web Services (AWS) 提供了多种云计算服务。

Amazon S3 (Simple Storage Service) 是一种对象存储服务，提供行业领先的可扩展性、数据可用性、安全性和性能。S3可以存储和检索任意数量的数据，常用于备份和恢复、归档、大数据分析和静态网站托管。

Amazon EC2 (Elastic Compute Cloud) 是一种提供可调整计算容量的web服务。它设计用于使开发人员能够进行云端web规模计算变得更加简单。EC2实例可以在几分钟内启动，并且可以根据需求进行扩展或缩减。

Amazon RDS (Relational Database Service) 使在云中设置、操作和扩展关系数据库变得简单。它提供经济高效且可调整大小的容量，同时自动执行耗时的管理任务，如硬件配置、数据库设置、补丁和备份。

Amazon Lambda 是一项无服务器计算服务，可运行代码而无需预置或管理服务器。Lambda自动运行代码以响应多个事件，如HTTP请求、S3桶中的对象修改、DynamoDB表更新等。

Amazon Bedrock 是一项全托管服务，提供来自领先AI公司的高性能基础模型。通过Bedrock，开发者可以轻松访问Claude、Llama、Titan等大型语言模型，无需自己管理基础设施。

Amazon CloudFront 是一种快速内容分发网络(CDN)服务，可安全地以低延迟和高传输速度向全球客户分发数据、视频、应用程序和API。

Amazon DynamoDB 是一种完全托管的NoSQL数据库服务，提供快速且可预测的性能以及无缝的可扩展性。它是一个键值和文档数据库，可以支持任何规模的应用程序。

Amazon SageMaker 是一项完全托管的服务，使开发人员和数据科学家能够快速轻松地构建、训练和部署机器学习模型。SageMaker消除了机器学习过程中的繁重工作。
